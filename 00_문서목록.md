# 프로젝트 문서 목록 (정본)

**이야기가 있는 여행 (Travel with Stories)** — K-Culture 종로·북촌 권역 + 콘텐츠 태그 모델.
모든 문서는 **권역 모델 + 일반화 근거**로 정렬됨.

## 1. 기획·설계 문서 (정본)
| 문서 | 역할 |
|---|---|
| `문제정의서_권역_정본.docx` | 문제·근거(일반화: 셋제팅·성지순례)·페르소나·범위(권역) |
| `서비스기획서_권역_정본.docx` | 컨셉(권역+태그)·기능 F1~F5·장소 매트릭스(7곳)·리스크·일정 |
| `서울_권역_스토리동선_AI봇_설계서.docx` | 화면 흐름·입출력·데이터 구조(Place 다태그) |

## 데모 권역 구성 (7장소 · 앱과 일치)
- 폭군의 셰프 → 소주방 · 경회루 · 운경고택(검토)
- 참교육/도깨비/겨울연가/여신강림 → 중앙고등학교 (교차 안내)
- 도깨비 → 북촌한옥마을
- 옷소매 붉은 끝동 → 창덕궁 후원
- 스트리트 푸드(넷플릭스) → 광장시장

## 메모
- 옛 per-content 문서·고궁 데이터 버전은 폐기함.
- ‘권역모델_기획뼈대’의 전환 논리는 서비스기획서 §2에 통합됨.
- 일부 docx가 마운트 동기화로 손상된 이력 있음 → 발표 전 PC에서 열림 확인 권장.
